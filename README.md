# Worker Synergy

Программа для работы с данными работников Университета «Синергия».  
Выполнена в рамках производственной практики (технологической, проектно-технологической).

## Описание

Класс `WORKER` хранит данные о работнике: ФИО, должность, зарплату и год поступления на работу.  
Программа позволяет вводить данные, просматривать список и фильтровать работников по стажу.

## Структура проекта

- `worker.py` — модуль с описанием класса Worker
- `main.py` — основная программа

## Запуск

```bash
python main.py
```

## Требования

- Python 3.8+

